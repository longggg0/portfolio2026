# 🌐 Horn Bunlong — Personal Portfolio 2026

A modern, responsive personal portfolio website built with React.js and Vite. Showcasing my skills, projects, and experience as a full-stack web development student.

## 🔗 Live Demo

[View Portfolio](https://mini-ecommerce-techstore-5v3b.vercel.app/)

---

## 🛠 Built With

- **React.js** — UI library
- **Vite** — Build tool
- **Tailwind CSS** — Utility-first styling
- **Lucide React** — Icon library
- **Telegram Bot API** — Contact form integration

---

## 📁 Project Structure
```
src/
├── assets/          # Images and static files
├── components/      # Reusable components
├── sections/
│   ├── About.jsx    # About me, education, experience
│   ├── Skills.jsx   # Technical skills & proficiency
│   ├── Projects.jsx # Project showcase
│   └── Contact.jsx  # Contact form via Telegram
└── App.jsx
```

---

## ✨ Features

- 📱 Fully responsive — mobile, tablet & desktop
- 🌙 Dark theme UI
- 📬 Contact form connected to Telegram Bot
- 🗂 Project showcase with live demo links
- 📊 Skill proficiency bars
- 🎓 Education & experience timeline

---

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/longggg0/portfolio2026.git

# Navigate into the project
cd portfolio2026

# Install dependencies
npm install

# Start the development server
npm run dev
```
